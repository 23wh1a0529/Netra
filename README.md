# 🚓 NETRA

## Smart Monitoring & Emergency Response System for Anantapur

> **NETRA** is a real-time, hierarchical monitoring and emergency response platform designed to modernize law enforcement operations in **Anantapur district**.

---

## 🎯 Problem Statement

In large districts like Anantapur:

* 🚫 Communication between officers is **slow & unstructured**
* 🚫 No **real-time visibility** of field personnel
* 🚫 Emergency alerts are **delayed or misrouted**
* 🚫 Lack of **hierarchical control** leads to misuse

---

## 💡 Our Solution — NETRA

NETRA introduces:

* ⚡ **Real-time officer tracking**
* 🔐 **Strict hierarchical communication system**
* 🚨 **Instant emergency alert forwarding**
* 🛑 **Bandobasthu mode for critical operations**

---

## 🌍 Real-World Impact (Anantapur Focus)

* 👮 Track officers live across zones
* 🚓 Improve response time during emergencies
* 🧭 Enable **district-level coordination**
* 🔒 Ensure communication flows **only through proper authority chains**

---

## 🚀 Key Features

### 🔐 Role-Based Hierarchy

* Admin → DSP → CI → SI → Constable
* ❌ Lower roles cannot access higher-level actions
* ✅ Secure, structured command system

---

### 📍 Live Location Tracking

* Real-time geo-tracking of officers
* Enables centralized monitoring

---

### 🚨 Emergency Alert System

* Send alerts within hierarchy
* Forward only to **higher authorities**
* Prevent unauthorized escalation

---

### 🛑 Bandobasthu Mode

* Special mode for:

  * Public events
  * High-risk situations
* Controlled alert broadcasting

---

### ⚡ Real-Time Updates

* Powered by WebSockets
* Instant communication across all users
